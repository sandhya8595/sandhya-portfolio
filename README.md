# 🌐 Personal Portfolio

# 👋 About

Welcome to my personal portfolio! This portfolio showcases my journey as a **Full Stack Developer** and **Generative AI Enthusiast**, highlighting my technical skills, projects, and achievements.

The interface was first designed in **Figma** and then transformed into a fully responsive web application using **Next.js** and **TypeScript**. The goal was to create a modern, interactive, and visually engaging experience while maintaining high performance and accessibility.

---

# ✨ Features

- Modern and interactive UI
- Fully responsive across all devices
- Smooth animations and page transitions
- About Me section
- Skills showcase
- Featured Projects
- Experience & Education
- Contact section
- Resume download
- Social media integration
- Fast loading and optimized performance

# ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

Move to the project directory

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
